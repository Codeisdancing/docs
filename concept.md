# TarotAI: AI Таролог с Коллекционированием

## Концепция приложения

### Основная идея
AI-powered таролог с геймифицированной системой коллекционирования скинов для карт, бустеров и кристаллов. Приложение сочетает психологическую привлекательность гаданий на Таро с вирусной механикой коллекционирования в стиле Labubu.

## Ключевые механики

### 1. AI Таролог
- **Персонализированные расклады**: AI анализирует вопросы пользователя и генерирует уникальные интерпретации
- **Множество типов раскладов**: от простых (3 карты) до сложных (Кельтский крест)
- **Голосовой ассистент**: AI-таролог с настраиваемым голосом и персоналией
- **История и аналитика**: отслеживание паттернов в раскладах пользователя

### 2. Система коллекционирования

#### Скины для карт
- **Базовые колоды**: классическая, минималистичная, неоновая
- **Редкие колоды**: аниме, киберпанк, фэнтези, зодиакальная
- **Эпические колоды**: голографическая, элементальная, космическая
- **Легендарные колоды**: персонализированная с AI-генерацией под пользователя

#### Рарность и дропы
- Common (60%)
- Uncommon (25%)
- Rare (10%)
- Epic (4%)
- Legendary (1%)

### 3. Кристаллы (внутренняя валюта)
- **Получение**:
  - Ежедневный вход: 10-50 кристаллов
  - Выполнение ежедневных раскладов: 20-100 кристаллов
  - Достижения: 50-500 кристаллов
  - Покупка за реальные деньги

- **Использование**:
  - Покупка бустеров
  - Разблокировка премиум раскладов
  - Покупка скинов напрямую
  - Активация бафов

### 4. Бустеры
- **Обычный бустер** (100 кристаллов):
  - 3 карты-скина
  - Гарантированно 1 Uncommon+
  
- **Редкий бустер** (300 кристаллов):
  - 5 карт-скинов
  - Гарантированно 1 Rare+
  - Шанс на баф

- **Эпический бустер** (1000 кристаллов):
  - 10 карт-скинов
  - Гарантированно 1 Epic+
  - 2-3 бафа
  - Мини-кристаллы (50-200)

### 5. Система бафов
- **Баф прозрения**: +50% детализации в интерпретациях на 24 часа
- **Баф удачи**: x2 кристаллы за активности на 12 часов
- **Баф коллекционера**: +20% шанс на редкие дропы на 3 часа
- **Баф мудрости**: разблокировка эксклюзивных раскладов на 7 дней
- **Баф резонанса**: AI лучше понимает контекст вопросов на 48 часов

### 6. Социальные механики
- **Гильдии тарологов**: объединения игроков с общими бонусами
- **Обмен скинами**: маркетплейс для обмена дубликатов
- **Турниры предсказаний**: соревнования на точность интерпретаций
- **Подарки друзьям**: дарение бустеров и кристаллов

### 7. Сезоны и события
- **Месячные сезоны** с тематическими коллекциями
- **Праздничные события** с эксклюзивными скинами
- **Боевой пропуск** с наградами за прогресс
- **Коллаборации** с известными художниками и брендами

## Техническая реализация

### Стек технологий
- **Frontend**: React Native (iOS/Android)
- **Backend**: Node.js + Express
- **AI**: OpenAI API / Claude API для интерпретаций
- **БД**: PostgreSQL для данных, Redis для кэша
- **Платежи**: Stripe / In-app purchases
- **Аналитика**: Mixpanel + Custom events

### MVP функционал
1. Базовый AI таролог с 3 типами раскладов
2. Система из 5 базовых колод скинов
3. Простая система кристаллов и бустеров
4. Ежедневные награды
5. Базовая монетизация (кристаллы + VIP)
